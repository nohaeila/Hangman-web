# Mon Projet HangmanWeb

Ce projet a pour objectifs de pouvoir jouer au hangman sur interface Web, le tout dans un style néon.

## Installation

1.Télécharger le ZIP contenant ce fichier.
2.Ouvrir le dossier avec votre application.
3.Lancer la commande go run .\main.go.
4.Ouvrir le localhost8080 en cliquant sur le lien ou sur votre navigateur.

## Fonctionnalités

- Possibiliter de choisir votre nom d'utilisateur.
- Possibiliter de choisir le nieveau de difficulté.
- Jouer au Hangman.
- Avoir un récapitulatif de la partie.
